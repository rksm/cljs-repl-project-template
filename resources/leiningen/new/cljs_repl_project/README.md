## Setup

1. Compile ClojureScript code

        lein do cljsbuild clean, cljsbuild once

2. Start a web server

        ./scripts/start-web-server.sh

3. Start a repl

        ./scripts/start-repl-server.sh

4. Connect to the repl and have fun!
